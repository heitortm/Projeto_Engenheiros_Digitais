Sistema de Chamados por Geolocalização
Descrição
Este projeto é um sistema de chamados voltado para o monitoramento e gerenciamento de incidentes com base em geolocalização. Os usuários podem registrar chamados associados a problemas em locais específicos, permitindo que a equipe de suporte visualize a localização e o status dos chamados em um mapa interativo.

Funcionalidades
Cadastro de Chamados: Os usuários podem registrar novos chamados com informações detalhadas como título, descrição e localidade.
Geolocalização: Utilizando O Leaflet, o sistema identifica automaticamente a localização que foi definida no cadastro para associar o chamado ao local correto.
Mapa Interativo: Os chamados são exibidos em um mapa interativo
Atualização de Status: A equipe de suporte pode alterar o status dos chamados (Aberto, Em andamento, Resolvido) e adicionar comentários.
Tecnologias Utilizadas
Angular: Estruturação das páginas web.
SCSS: Estilização e layout do projeto.
TypeScript: Interatividade e manipulação da API do mapa.
Leaflet.js: Biblioteca JavaScript para exibição de mapas interativos.
Back-end (Futuro): O projeto poderá ser integrado a um back-end usando Node.js e bancos de dados como MongoDB para armazenamento de chamados.
